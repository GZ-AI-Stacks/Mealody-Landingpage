KURZFASSUNG

Mealody ist eine Offline-App. Sie kommt ohne Konto, ohne Login und ohne eigenen Server aus. Alles, was du eingibst (eigene Rezepte, Einkaufslisten, Design-Wahl), bleibt ausschließlich lokal in deinem Browser. Wir erheben, speichern und übertragen keine personenbezogenen Daten auf einen Server von uns.

2. Lokale Speicherung im Browser (localStorage)

Mealody speichert deine Eingaben im sogenannten localStorage deines Browsers. Das ist ein Speicherbereich auf deinem eigenen Gerät. Gespeichert werden:

  • von dir selbst angelegte Rezepte,
  • deine Einkaufslisten samt eingetragener Preise,
  • deine gewählte Stimmung und das gewählte Design (Theme),
  • die Zoom-Einstellung.

Diese Daten verlassen dein Gerät nicht. Sie werden nicht an uns oder an Dritte übertragen. Du kannst sie jederzeit selbst löschen, indem du die Daten der Website in deinen Browser-Einstellungen entfernst (Browserdaten / Websitedaten löschen).

3. Keine Cookies, kein Tracking

Mealody setzt keine Tracking-Cookies, nutzt keine Analyse-Werkzeuge (z. B. Google Analytics) und erstellt keine Nutzerprofile.

4. Keine externen Inhalte (CDN-frei)

LOKAL EINGEBETTET

Die Schriftart "Inter" und die Icon-Sammlung "Font Awesome" sind vollständig lokal in der App enthalten. Mealody lädt keinerlei Inhalte von externen Servern (Content Delivery Networks) nach – es wird keine IP-Adresse an Dritte übertragen. Die App funktioniert vollständig offline.
